АРМ сервисного инженера по обслуживанию домофона многоквартироного дома.

Функциональные требования:
Необходимо регистрировать заявки на обслуживание домов для каждого инженера.
Должна быть возможность заполнения списка районов, улиц и домов.
Каждый сервисный инженер обслуживает только определенные дома.
В заявке указывается выполненная услуги и его цена.
Список оказываемых услуг определен заранее.

Технические требования:
Локальная база данных - SqLite.
ORM для доступа к БД - Dapper.
Web-клиент - ASP.Net MVC

Примечание:
Тестовая БД (test.sqlite) находится в директории \dispatchservice.web\App_Data
Структура БД описана в файлах DbShema.png и Db.db.
